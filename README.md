# Clone this Repo

## git clone https://github.com/arjun7736/Restaurant-Management.git

# Navigate to the folder

## cd Restaurant-Management

## npm install

## touch .env

# Add the following variables to the .env file
## PORT=3000
## MONGO_URI= "MONGO URI"

# Run the application (Both  Frontend and Backend)
## npm run dev
